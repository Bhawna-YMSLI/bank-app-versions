# Bank App Frontend

Angular frontend for the bank management system.

## Features
- JWT-based login and logout.
- Role-based routing and authorization guard.
- Manager workflows: account management, clerk management, pending withdrawal approvals, transaction history.
- Clerk workflows: deposit, withdrawal, transaction history.
- Centralized unauthorized handling and user-friendly API error messages.

## Configuration
- API base URL is configured in `src/environments/environment.ts`.

## Run
```bash
npm install
npm start
```
